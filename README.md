# <p align="center">🎵 MDfy — Music Without Limits</p>

**MDfy** (Music + Modify) — это амбициозный музыкальный плеер для Android с открытым исходным кодом. Проект создан для тех, кому мало просто слушать музыку — здесь ты можешь настраивать всё, от кроссфейда до поведения частиц визуализатора.

---

## 🚀 Что делает MDfy особенным?

* **Живой интерфейс:** Продвинутая система визуализации на основе динамических частиц и плавных спектров (как ты видишь выше!).
* **Гибридный движок:** Ищет метаданные и обложки через Spotify API, а поток берет из YouTube.
* **Полная свобода:** Скачивай треки с автоматической пропиской ID3-тегов и обложек высокого качества.
* **Clean Architecture:** Код написан по канонам современной Android-разработки (Kotlin, Jetpack Compose, Hilt, Coroutines).

---

## 🛠 Технологии

* **UI:** Jetpack Compose (Material 3 + Dynamic Colors)
* **Core:** Media3 ExoPlayer для безупречного звука
* **Data:** Room, DataStore Preferences, Retrofit
* **DI:** Hilt (Dependency Injection)
